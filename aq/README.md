# aq
deCrypter for Anglobal communications
```
USAGE:
    aq [FLAGS] [QUERY]

FLAGS:
    -h, --help       Prints help information
    -i               start interactive prompt
    -t               prints hex trinomes
    -V, --version    Prints version information

ARGS:
    <QUERY>    an alphanumeric-encoded string
```
